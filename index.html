自由软件/开源软件

开源软件（英语：open source software，缩写：OSS）又称开放源代码软件，是一种源代码可以任意获取的计算机软件，这种软件的著作权持有人在软件协议的规定之下保留一部分权利并允许用户学习、修改以及以任何目的向任何人分发该软件。开源协议通常符合开放源代码的定义的要求。一些开源软件被发布到公有领域。开源软件常被公开和合作地开发。开源软件是开放源代码开发的最常见的例子，全球闻名的 Linux,Android等都是开源软件。

加入我们

长沙 Linux 用户组（英文 Changsha GNU/Linux Group，简称 ChangshaLUG） 是长沙本地的 Linux 用户组，旨在促进开源软件的交流和沟通。TJLUG 是一个完全自由的非盈利开源社区，鼓励和欢迎关于开源和开源软件的交流。 我们讨论的话题主要围绕着 Linux、Operating System、Kernel、和 Open Source，当然也可以是关于 Linux 以及开源的招聘和分享等相关信息。过去比较常见的话题是： 各种 Linux 发行版、 Kernel 及各种内核、 GNOME、KDE、GNU 及各种开源软件等等。 长沙 GNU/Linux 用户组是一个完全自由的非盈利开源社区，欢迎加入我们的线上和线下交流。
